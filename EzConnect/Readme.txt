<----------------------	EzConnect ---------------------->
1. Create war file for EzConnectPortal (using Eclipse)
2. Run EzConnectPortal.war on Tomcat server(put the war in webapp folder and run bin/startup)
3.Install generated apk file from android project EzConnectAndroid.
4.Register the app with server by entering proper ip Address of server on app UI.
5.Enter interviewer and interviewee number on portal to make call.
6.View report generated
